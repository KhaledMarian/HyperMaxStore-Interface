# Hyper Max Store - Application

**BTEC Database Assignment - Hyper Max Store Snacks and Chips Module**

Al-Ahliyya Amman University - 2026

## About

This repository contains the Windows Forms application for managing the Snacks and Chips department at Hyper Max Store. The application connects to a Microsoft SQL Server database to manage products, customers, and sales.

## How to Run

1. Download the files from this repository
2. Make sure you have Microsoft SQL Server installed
3. Set up the HyperMax_Snacks_And_Chips_Module database
4. Double-click `HyperMaxStore.exe` to launch the application

## Features

- Add, view, update, and delete products
- User role-based access control
- Data validation
- Reports for stock and sales

## Requirements

- Windows 10 or 11
- .NET Framework 4.7.2 or higher
- Microsoft SQL Server 2019 or higher
